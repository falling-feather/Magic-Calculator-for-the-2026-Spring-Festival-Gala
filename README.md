# 魔术计算器

这是根据2026马年春晚中邓男子表演的魔术《惊喜时刻》制作的道具网页。网页模拟了一个手机计算器，内置了魔术机制，能够自动将输入的数字替换为与当前时间相关的数字。

## 用法

打开 `index.html` 文件即可使用。计算器可以正常进行各种运算。

要触发魔术效果，需要按以下步骤操作：

1. 打开页面或按 `AC` 清空
2. 连续点击 `=` 等号键若干次（比如 3 次）
3. 输入一个 7 位或以上的数字（比如 `1234567`）
4. 点击 `+` 或 `−` 运算符
5. 继续点击运算符，当点击次数达到之前等号键点击次数减 1 时，数字会自动替换为与当前时间相关的数字

## 示例

假设现在是 1 月 15 日 10 点 30 分：

```
1. 打开页面
2. 连续按 3 次 "="
3. 输入：1234567
4. 按 "+"（第 1 次）
5. 再按 "+"（第 2 次，此时触发替换）
6. 数字自动变为与当前时间相关的数字
```

时间数字格式是：月 + 日 + 小时 + 分钟。例如 1 月 15 日 10 点 30 分对应 `1151030`。

## 注意事项

- 魔术替换只在 `+` 和 `−` 运算符下生效
- 输入数字需要达到 7 位或以上
- 按 `AC` 会重置所有状态
